== Description ==

Introducing a quick and easy to use solution for selling product downloads on your self hosted WordPress website or blogging platform. Using the WP E-Junkie Shopping Cart system, all you need is an E-Junkie account and your E-junkie client id. The plugin is totally free and open source. Custom buttons can even be added via the plugin settings page!

Buttons are easily added to a post/page by pasting a small shortcode containing the product id of any product hosted on E-junkie.

Not sure if E-junkie is right for you? Get a 60 day free trial when you download and install the plugin.

== Installation ==

1. After unzipping, upload everything in the `wp-ejunkie` folder to your `/wp-content/plugins/` directory (preserving directory structure).

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Go to the WP E-junkie options page under the settings tab and enter your E-junkie ID.

4. Go to the Cart Items page to add your items from E-junkie.

5. Add `[EJUNKIE_ADD2CART item="your item id"]` and `[EJUNKIE_VIEWCART]` to a post or page to display your E-Junkie buttons.
